# 📚 Books Store (Django Project)

This is a simple online Book Store web application built using Django.

## Features
- View books
- Search books
- Add to cart
- Remove from cart
- Cart total amount
- Buy now option
- Payment page
- About page
- Contact page

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## How to Run

1. Clone the repository
2. Open project folder
3. Run the command:

python manage.py runserver

4. Open browser:

http://127.0.0.1:8000
